Armazenamento de dados

Armazene e acesse seus dados no GEO-SIG, usando métodos flexíveis que se adaptam aos seus fluxos de trabalho. Use a nuvem, formatos de arquivo comuns ou o geodatabase (método da Esri para armazenar e gerenciar dados geoespaciais).

Avalie e valide os dados

O GEO-SIG contém um conjunto completo de ferramentas para verificar e validar a integridade dos dados. Os conjuntos de ferramentas de validação verificam arquivos de terceiros e executam processos de reparo de dados selecionados para avaliar os dados e garantir alta qualidade.
